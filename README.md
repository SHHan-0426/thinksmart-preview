# 씽크스마트 v3 시험판 (Preview)

- **라이브**: https://thinksmart-preview.netlify.app
- **대상**: 김태영 대표(씽크스마트) 검수용 시험판
- **제안자**: 거닐만 한성호 PM (`findflow@hanmail.net`)

## 운영 매뉴얼

### 신규 게시물 반영
1. Meta 계정 센터에서 인스타 백업 zip 다운로드 (월 1회)
2. 한성호 PM에게 전달 → 자동 추출 후 푸시

### 사이트 수정 → 라이브 반영
```bash
git add .
git commit -m "내용"
git push origin main
# → Netlify 1~2분 내 자동 반영
```

### 행사 추가 (수동 큐레이션)
`data/events.json` 편집 → `instagram_url` 필드에 인스타 게시물 URL 입력

## 기술

- 정적 HTML/CSS/JS (Vanilla, 빌드 도구 없음)
- 모바일 퍼스트 (최대 420px 프레임, 데스크탑에서도 핸드폰 시뮬레이션)
- 데이터: JSON (저자·도서·이벤트·인스타)
- 배포: Netlify (main 브랜치 자동 배포)
